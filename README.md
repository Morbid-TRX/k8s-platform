# k8s-platform

A production-grade, multi-tenant Kubernetes platform built for learning and portfolio purposes.
Demonstrates real DevOps/SRE skills — GitOps, observability, security, and scalability.

## Stack
- **Runtime**: kind (local) → AWS EKS (cloud)
- **GitOps**: ArgoCD
- **Observability**: Prometheus + Grafana + Loki
- **Security**: Kyverno + pre-commit + gitleaks
- **CI/CD**: GitHub Actions
